MODALIDAD DE PROGRAMACION Y DISEÑO WEB

Seccion 1006

La modalidad de programación y diseño web se centra en la creación y desarrollo de sitios y aplicaciones para internet, combinando habilidades técnicas y creativas. Por un lado, la programación web permite que las páginas funcionen correctamente mediante lenguajes como HTML, CSS y JavaScript, tanto en la parte visible (frontend) como en la lógica interna (backend). Por otro lado, el diseño web se enfoca en la apariencia, la organización del contenido y la experiencia del usuario, buscando que los sitios sean atractivos, fáciles de usar y adaptables a diferentes dispositivos. Esta modalidad suele ser práctica, basada en proyectos, y prepara a las personas para crear soluciones digitales útiles en distintos ámbitos.

¿Que es una lista ordenada?

Es un grupo de elementos organizados de una manera especifica donde cada elemento tiene un marcador que indica su posicion

Ejercicio practico de lista ordenada

  1. mandarinas
  2. manzanas
  3. uvas
  4. fresas
  5. piñas

¿Que es una lista no ordenada?

Es un grupo de elementos que no necesitan un orden especifico y se marcan con puntos negros o algun otro caracter

Ejercicio practico de lista no ordenada

  • coger una taza
  • servir agua
  • calentar el agua
  • volver a servir el agua
