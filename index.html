¿Que es una lista ordenada?

Es un grupo de elementos organizados de una manera especifica donde cada elemento tiene un marcador que indica su posicion

Ejercicio practico de lista ordenada

  1. mandarinas
  2. manzanas
  3. uvas
  4. fresas
  5. piñas

¿Que es una lista no ordenada?

Es un grupo de elementos que no necesitan un orden especifico y se marcan con puntos negros o algun otro caracter

Ejercicio practico de lista no ordenada

  • coger una taza
  • servir agua
  • calentar el agua
  • volver a servir el agua
